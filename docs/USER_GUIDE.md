## ğŸ‘¨â€ğŸ’¼ Untuk Admin

### Hak Akses Admin
- Manajemen user (buat, edit, hapus)
- Manajemen departemen dan kelas
- Manajemen kurikulum (KI/KD, ATP, Modul, Deep Learning)
- Verifikasi journal mengajar
- Lihat laporan dan statistik
- Manajemen izin/absensi

### Tugas-Tugas Utama

#### 1. Manajemen User
```
Menu > User Management
```
- Buat user baru (Guru, Siswa, Kepala Sekolah)
- Edit profil user
- Deaktifkan/aktifkan user
- Reset password user

#### 2. Manajemen Kelas
```
Menu > Master Data > Kelas
```
- Buat kelas baru (X TKJ A, XI TKJ B, XII TKJ C)
- Assign wali kelas
- Assign siswa ke kelas
- Lihat daftar siswa per kelas

#### 3. Manajemen Kurikulum
```
Menu > Kurikulum
```

**KI/KD (Kurikulum 2013)**:
- Buat Kompetensi Inti (KI) per kelas
- Buat Kompetensi Dasar (KD) per KI
- Edit/hapus KI/KD

**ATP (Kurikulum Merdeka)**:
- Buat Alur Tujuan Pembelajaran
- Tentukan fase pembelajaran (D, E, F)
- Set durasi pembelajaran

**Modul Pembelajaran**:
- Buat modul baru
- Link ke ATP
- Upload file modul
- Set difficulty level (Mudah, Sedang, Sulit)

**Deep Learning**:
- Buat program pembelajaran mendalam
- Tentukan 3 fase (Problem Understanding, Knowledge Construction, Application)
- Tentukan untuk kelas berapa

#### 4. Verifikasi Journal Mengajar
```
Menu > Journal > Pending Verification
```
- Lihat journal yang menunggu verifikasi
- Baca detail journal
- Approve atau kembalikan untuk revisi
- Tambahkan komentar

#### 5. Dashboard Admin
```
Dashboard
```
- Lihat statistik user
- Lihat statistik journal (bulan ini, pending)
- Lihat attendance rate keseluruhan
- Lihat recent activities
- Lihat pending permissions

## ğŸ‘¨â€ğŸ« Untuk Guru

### Hak Akses Guru
- Buat dan kelola journal mengajar
- Catat absensi siswa
- Lihat data siswa dan kelas
- Lihat kurikulum (KI/KD, ATP, Modul)
- Lihat dashboard

### Tugas-Tugas Utama

#### 1. Buat Journal Mengajar
```
Menu > Journal > Buat Journal Baru
```

**Langkah-langkah**:
1. Pilih kelas yang akan diajar
2. Pilih tanggal pembelajaran
3. Isi mata pelajaran/topik
4. Pilih KI/KD yang relevan (optional)
5. Pilih modul pembelajaran (optional)
6. Isi detail pembelajaran:
   - Ringkasan materi yang diajarkan
   - Metode pembelajaran (Ceramah, Diskusi, Praktik, dll)
   - Aktivitas pembelajaran
   - Jumlah siswa hadir
7. Isi penilaian:
   - Metode assessment
   - Catatan pencapaian
   - Catatan kendala/hambatan
8. Isi media dan bahan referensi
9. Isi rencana tindak lanjut
10. Klik "Simpan" atau "Simpan & Submit"

**Tips**:
- Jangan lupa submit journal untuk approval admin
- Lengkapi semua field untuk journal yang lebih informatif
- Cantumkan KI/KD dan modul untuk tracking kurikulum

#### 2. Edit Journal Mengajar
```
Menu > Journal > Daftar Journal > Klik Edit
```
- Edit journal yang belum disubmit
- Catatan: Journal yang sudah disubmit tidak bisa diedit (harus permintaan admin)

#### 3. Submit Journal
```
Menu > Journal > Daftar Journal > Klik Submit
```
- Submit journal untuk approval admin
- Journal tidak bisa diedit setelah submit

#### 4. Catat Absensi
```
Menu > Attendance > Catat Absensi
```

**Pilihan 1: Catat Per Siswa**
- Pilih kelas
- Pilih tanggal
- Input absensi setiap siswa
- Status: Hadir, Sakit, Izin, Alfa

**Pilihan 2: Catat Massal**
- Pilih kelas
- Pilih tanggal
- Buka CSV template
- Isi status untuk setiap siswa
- Upload file CSV

#### 5. Lihat Data Siswa
```
Menu > Data Siswa > Pilih Kelas
```
- Lihat daftar siswa
- Lihat detail siswa
- Lihat absensi siswa
- Lihat riwayat izin

#### 6. Dashboard Guru
```
Dashboard
```
- Lihat ringkasan journal bulan ini
- Lihat jumlah journal yang pending approval
- Lihat kelas yang dibimbing
- Lihat attendance rate per kelas
- Lihat recent activities

#### 7. Lihat Kurikulum
```
Menu > Kurikulum
```
- Lihat KI/KD untuk referensi mengajar
- Lihat ATP dan learning objectives
- Download modul pembelajaran
- Lihat program deep learning

### Tugas-Tugas Utama

#### 1. Lihat Absensi Pribadi
```
Menu > Absensi > Absensi Saya
```
- Lihat history absensi perhari

#### 2. Ajukan Surat Izin
```
Menu > Izin > Ajukan Surat Izin
```

**Langkah-langkah**:
1. Klik "Ajukan Izin Baru"
2. Isi informasi:
   - Tanggal mulai
   - Tanggal akhir
   - Alasan izin (Sakit, Izin Orang Tua, Acara, dll)
   - Tipe izin
3. (Optional) Upload surat izin/surat keterangan
4. Klik "Ajukan"

**Status Izin**:
- **Pending**: Menunggu approval guru/admin
- **Approved**: Izin disetujui
- **Rejected**: Izin ditolak

**Tips**:
- Sertakan bukti (surat dokter, surat orang tua) untuk lebih cepat diapprove
- Ajukan izin sebelum atau segera setelah ketidakhadiran

#### 3. Lihat Informasi Kelas


- Lihat jadwal pembelajaran

#### 4. Akses Modul Pembelajaran
```
Menu > Modul > Download Modul
```
- Lihat daftar modul per kelas
- Download materi pembelajaran
- Akses deep learning resources

#### 5. Dashboard Siswa

## â“ FAQ

### Q: Bagaimana jika lupa password?
**A**: 
1. Klik "Lupa Password" di login page
2. Masukkan email
3. Cek email untuk reset link
4. Buat password baru
5. Login dengan password baru

### Q: Bagaimana jika journal tertolak admin?
**A**: 
1. Lihat feedback dari admin
2. Edit journal sesuai feedback
3. Resubmit journal

### Q: Bisakah menghapus journal yang sudah disubmit?
**A**: Tidak. Journal yang sudah disubmit tidak bisa diedit atau dihapus. Hubungi admin jika perlu revisi.

### Q: Bagaimana format upload surat izin?
**A**: 
- Format: PDF, JPG, PNG
- Ukuran maksimal: 10 MB
- Pastikan terlihat jelas dan lengkap

### Q: Kapan attendance summary di-finalize?
**A**: 
- Biasanya akhir bulan
- Admin akan finalize dan lock
- Setelah di-lock tidak bisa diubah

### Q: Bagaimana jika ada kesalahan absensi?
**A**: 
1. Hubungi guru atau admin
2. Admin bisa revisi/perbaiki
3. Attendance summary akan diupdate

### Q: Apa itu KI/KD?
**A**: 
- **KI** = Kompetensi Inti (kompetensi level tinggi)
- **KD** = Kompetensi Dasar (penjabaran spesifik dari KI)
- Dari Kurikulum 2013

### Q: Apa itu ATP?
**A**: 
- **ATP** = Alur Tujuan Pembelajaran
- Tujuan pembelajaran yang terstruktur bertahap
- Dari Kurikulum Merdeka

### Q: Bisa lihat journal guru lain?
**A**: 
- Guru hanya bisa lihat journal mereka sendiri
- Admin & Kepala Sekolah bisa lihat semua
- Siswa tidak bisa lihat journal

### Q: Bagaimana jika ada technical issue?
**A**: 
1. Hubungi IT admin sekolah
2. Sertakan error message
3. Beritahu apa yang Anda lakukan saat error

### Q: Berapa lama persetujuan izin?
**A**: 
- Biasanya 1 hari kerja
- Tergantung kesibukan guru/admin
- Hubungi jika lebih dari 2 hari belum disetujui
